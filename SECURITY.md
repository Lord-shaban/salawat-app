# Security Policy

## Supported Versions

At this moment, only the **latest `main` branch version** of the Salawat App is supported with regular updates and security patches.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| <= v0.x | :x:                |

## Reporting a Vulnerability

We take the security of this application very seriously, as it serves as a robust platform for global users. If you discover a vulnerability, please do NOT report it by opening a public Issue on GitHub.

Instead, please send an email or contact the maintainers privately (or open a Draft Security Advisory if available on our GitHub Security tab).

1. Report the vulnerability with detailed steps to reproduce.
2. We will analyze the submission and verify the vulnerability.
3. If confirmed, a patch will be prepared and published as quickly as possible.
4. We will publicly acknowledge your responsible disclosure upon resolving the issue.

Thank you for helping us keep Salawat App safe and robust.
