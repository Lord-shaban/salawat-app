# Support

Thank you for using and contributing to **Salawat App**. We are dedicated to building a supportive and transparent global community.

## How to get help

If you are experiencing issues or have questions regarding the application, please explore the following avenues:

### 1. Read the Documentation
Before asking for help, please verify that your question hasn't already been answered in our documentation:
- [README.md](./README.md) - For installation and general overview.
- [ARCHITECTURE.md](./ARCHITECTURE.md) - For understanding how the system runs under the hood.

### 2. Search Existing Issues
Your issue might have been reported or resolved already. Please search the [GitHub Issues](https://github.com/your-username/salawat-app/issues) before opening a new one.

### 3. Open a New Issue
If you cannot find an existing issue that addresses your problem:
- **For Bugs:** Open an issue using the `[BUG]` template. Provide logs, your OS, browser, and steps to reproduce.
- **For Features:** Open an issue using the `[FEATURE]` template.

### 4. Security Vulnerabilities
Do NOT open an issue for security vulnerabilities. Please report them privately by reading our [SECURITY.md](./SECURITY.md) policy.

## Community

As the project grows, we will introduce dedicated community spaces (e.g., Discord, Slack, or GitHub Discussions). For now, GitHub Issues is our primary hub for all support and community engagement.
